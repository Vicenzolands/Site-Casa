Display LCD módulo I2C

O display de cristal líquido (LCD) é uma tela fixa, usada para transmitir informações visuais (textos, imagens e vídeos), por meio eletrônico. Existe dois tipos comum de Display LCD: Display LCD 16x2(16 colunas e 2 linhas) e o outro é 20x4. Modulo serial I2C é um adaptador para o display usado para economizar pinos que serão usados no Arduino.
